# WeiboSDKExample #
新浪微博SDK示例，基于v2版API接口，对认证和请求进行了封装，包含最新微博列表、多帐号管理、发布文字微博、发布图片微博等功能示例。

##参考文档##
关于API的完整文档请参考：[新浪微博API文档](http://open.weibo.com/wiki/%E9%A6%96%E9%A1%B5 "新浪微博API文档") 。
